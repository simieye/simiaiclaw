# PixieDream Studio · 商业广告级AI修图智能体

> **一句话定位**：全链路AI自动化，一键输出摄影师审美标准的商业广告级成片。

---

## 一、产品概述

### 官网入口
https://pixiedream-studio.lovable.app/

### 核心定位

PixieDream Studio 是面向**商业内容生产场景**的AI图像精修工具，依托 AI Agent + Flux + ControlNet + LoRA 多技术栈协同，一站式完成从原图到广告级成片的全流程输出。

### 技术架构

| 技术 | 作用 |
|------|------|
| **AI Agent** | 全流程自动化编排，感知用户需求后自动调度各模块 |
| **Flux** | 高保真图像生成，保留细腻光影与材质质感 |
| **ControlNet** | 精准控制人物姿态、构图、光影，保障主体一致性 |
| **LoRA** | 自定义风格模型，锁定品牌专属视觉语言 |

---

## 二、核心能力

### 1. 全链路自动化

> 用户只需上传图片 + 描述需求，AI Agent 自动完成：抠图 → 生图 → 放大 → 合成 → 输出成片

无需分步操作，无需手动调整，一键完成全流程。

### 2. 商业级出图品质

输出成片符合**专业摄影师审美标准**，达到：
- 广告投放要求（300 DPI+）
- 品牌官网使用标准
- 印刷品输出质量

### 3. 亚像素级人像处理

| 处理维度 | 效果 |
|---------|------|
| 肤质 | 自然保留，无过度磨皮 |
| 发丝 | 根根分明，边缘精准 |
| 妆容 | 保留质感，无色块 |
| 光影 | 立体自然，无塑料感 |

### 4. 服装面料精准还原

支持各类材质的真实质感还原：
- **丝绸**：光泽流动感
- **牛仔**：粗犷纹理
- **皮革**：光泽与褶皱
- **针织**：蓬松绒感
- **纱质**：半透明层次

### 5. AI智能置景

将主体自动置入任意目标场景，AI 自动匹配：
- 光影方向（主光/补光/轮廓光）
- 透视关系（近大远小）
- 背景虚化程度（景深控制）

### 6. 自定义LoRA风格锁定

上传品牌专属 LoRA 模型，实现：
- 跨项目风格统一
- 跨批次批量生产
- 品牌视觉语言固化

## 四、使用教程

### Step 1：访问工具

```
打开：https://pixiedream-studio.lovable.app/
进入工作台，准备上传素材
```

### Step 2：上传原始图片

```
支持格式：JPG / PNG / WEBP
建议分辨率：不低于 1024×1024
上传后确认主体清晰、边缘干净
```

### Step 3：描述需求

```
描述方式建议：
✅ 详细版："时尚杂志风格，室外自然光，浅景深，背景虚化，妆容自然高级感"
✅ 简洁版："商业广告风格，户外场景，柔和光影"
✅ 参考版："参考[具体品牌/艺人风格描述]的色调和质感"

避免：
❌ 模糊描述："修好看点"
❌ 中文混杂："加个光影 please"
```

### Step 4：选择/上传LoRA（可选）

```
已有品牌LoRA → 直接应用
无LoRA → 使用平台默认风格
新创建 → 上传风格参考图训练
```

### Step 5：自动处理 → 下载成片

```
AI Agent 自动编排全流程
处理完成后可预览多版本
下载高分辨率原图（适配印刷/数字）
```

## 五、商业修图专业规范

### 质量标准参考

| 使用场景 | 最低分辨率 | 色彩模式 | 输出格式 |
|---------|-----------|---------|---------|
| 社交媒体 | 1080×1080 | sRGB | JPG 80%+ |
| 品牌官网 | 2400×2400 | sRGB | PNG |
| 印刷海报 | 3000×3000 | CMYK | TIFF |
| 户外广告 | 4000×4000+ | sRGB | PNG |

### 品牌视觉一致性检查

```
✅ 人物肤色统一（多图时肤色一致）
✅ 品牌色系统一（主色调偏差 < 5%）
✅ 光影风格统一（主光方向一致）
✅ 修图风格统一（统一使用同一LoRA）
```

### 明星/艺人修图特殊要求

```
⚠️ 肖像权：确认使用授权后方可商用
⚠️ 真实性：广告法要求不得过度美化失真
⚠️ 品牌调性：艺人形象需符合品牌定位
⚠️ 肤色自然：避免过度美白/磨皮，尊重艺人特征
⚠️ 多图一致性：同一艺人多图需保持特征连贯
```

## 七、避坑指南

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 上传模糊低分辨率原图 | 输出成片质量上限低 | 优先上传高质量原图（≥2K） |
| 过度依赖AI，不检查细节 | 光影穿帮、肤色不均 | AI处理后手动检查关键细节 |
| 批量处理不设置一致性参数 | 同一艺人多图风格割裂 | 统一LoRA + 统一参数模板 |
| 商用未确认肖像权 | 法律风险 | 使用前确认艺人/模特授权 |
| 不保存工程文件 | 无法二次修改 | 下载AI生成中间文件备份 |
